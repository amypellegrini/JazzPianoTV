# JazzPianoTV [![Build Status](https://travis-ci.org/amypellegrini/JazzPianoTV.svg?branch=dev)](https://travis-ci.org/amypellegrini/JazzPianoTV)

## Tech stack

- Node.js
- Express
- React
- Webpack
- Yarn
- Tape

## Commands

- Install: `yarn install`
- Test: `npm test`
- Build: `npm run build`
- Launch server: `npm start`
